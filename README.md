- [x] Create account

- [x] Unfollow User
- [x] See user profile
- [ ] See MY profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
